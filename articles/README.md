# 文字アフィリエイト 記事製造ライン

実際に使って良かった商品のURLと感想を渡すだけで、規約準拠の記事を生成・投稿するための仕組みです。

## 使い方（あなたがやること）

1. **商品URL（アフィリエイトリンク）** と **使ってみた感想** をチャットで私（Claude）に渡す
   - 例:「このモニターアーム、URLは https://amzn.to/xxxx 、半年使って肩こりが減った。組み立ては10分くらい。安っぽさはない。」
2. 私が `skills/affiliate-context/SKILL.md` と `articles/TEMPLATE.md` に沿って記事を生成
3. `articles/YYYY-MM-DD-<商品名>.md` として保存・コミット
4. 投稿先が自動化対応なら、`scripts/` のスクリプトで下書き投稿

## あなたが最初に1回だけやること

`skills/affiliate-context/SKILL.md` の「未記入」部分（プロフィール・ペルソナ・ジャンル）を
埋めてください。チャットで教えてくれれば私が埋めます。

## ディレクトリ構成

```
skills/affiliate-context/SKILL.md   # 永続コンテキスト（毎回読む前提情報）
articles/TEMPLATE.md                # 記事テンプレート（規約準拠の構成）
articles/YYYY-MM-DD-*.md            # 生成された記事
scripts/                            # 投稿スクリプト（プラットフォーム確定後に追加）
```

## 守っているルール（重要）

- **使った商品しかおすすめしない**（ステマ・景表法対策）
- **PR表記 / Amazonアソシエイト表記を必ず入れる**（2023年10月〜のステマ規制）
- 健康・美容・金融ジャンルは薬機法・景表法に配慮した表現に

# YouTube Shorts × アフィリエイト 収益化戦略

このドキュメントは、本リポジトリの YouTube Shorts 自動生成ツールを活用して
アフィリエイト収益を得るための戦略をまとめたものです。

---

## 1. 全体像（収益の仕組み）

```
Shorts動画で興味を引く
        ↓
概要欄 / 固定コメント / プロフィールリンク に誘導
        ↓
アフィリエイトリンク経由で商品購入・登録
        ↓
報酬発生
```

Shorts は直接リンクが踏まれにくいため、
**「リンクを踏みたくなるジャンル選び」** が収益を左右します。

---

## 2. ジャンル選定（最重要）

本ツールは「テキスト → 動画自動生成」が得意なため、
**テキスト主体で量産できるジャンル**が相性抜群です。

| ジャンル | 相性 | 主なASP / プログラム |
|---|---|---|
| 書籍・教養の要約 | ◎ 量産しやすい | Amazonアソシエイト、Audible |
| ガジェット / 便利グッズ | ◎ 単価高め | Amazon、楽天アフィリエイト |
| アプリ / サブスク紹介 | ○ | A8.net、もしもアフィリエイト |
| 金融 / 副業 / 学習 | ○ 単価高・審査厳しめ | A8.net、afb |

**推奨スタート: 書籍要約 or 便利グッズランキング**
（テキスト構成で動画化しやすく、Amazonアソシエイトと直結できるため）

---

## 3. ASP（アフィリエイトサービス）登録手順

1. **Amazonアソシエイト** — https://affiliate.amazon.co.jp/
   - 注意: 登録後 **180日以内に3件の売上**が無いと審査落ち。最初は手動でも実績を作る。
2. **楽天アフィリエイト** — 楽天会員ならすぐ利用可。審査ゆるめ。
3. **A8.net** — 案件数最大。アプリ/サブスク/金融系を扱うなら必須。
4. **もしもアフィリエイト** — Amazon/楽天をまとめて扱え、振込ハードルが低い。

---

## 4. 投稿フロー（量産の型）

1. ジャンル内でテーマを決める（例: 「読書術の本5選」）
2. 本ツールで動画を生成（`python main.py --topic "..." --duration 40`）
3. 概要欄テンプレートを付与（下記）
4. 固定コメントにもリンクを設置
5. 投稿 → 反応の良いテーマを横展開

### 概要欄テンプレート例

```
📚 動画で紹介した商品はこちら
▶ 商品A: https://amzn.to/xxxxx
▶ 商品B: https://amzn.to/yyyyy

※本動画はプロモーション（PR）を含みます。
※Amazonのアソシエイトとして適格販売により収入を得ています。
```

---

## 5. 法規制・ポリシー上の必須事項

- **ステマ規制（2023年10月〜）**: 広告には必ず「PR」「広告」を明記する（法的義務）。
- **Amazonアソシエイト規約**: 「Amazonのアソシエイトとして〜収入を得ています」の文言を表示。
- **YouTubeポリシー**: AIナレーション丸読みの大量投稿は
  「再利用されたコンテンツ / スパム」と判定され収益化剥奪・BANのリスク。
  → 1本ごとに独自の構成・編集・付加価値を入れる設計が必須。

## 7. ロードマップ（目安）

| フェーズ | 期間 | ゴール |
|---|---|---|
| 1. 立ち上げ | 〜2週間 | ジャンル確定、ASP登録、手動で5本投稿し売上実績を作る |
| 2. 自動化 | 〜1ヶ月 | ツールの動画生成・アップロードを実装し半自動運用 |
| 3. 拡大 | 〜3ヶ月 | 反応の良いテーマを量産、複数ASPで単価最適化 |

---

> 補足: 短期間で大きく稼ぐより、規約を守りつつ「特化 × 継続」で
> チャンネルを資産化するのが、結果的に最も収益が安定します。
